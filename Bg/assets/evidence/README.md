# Evidence Assets

This folder contains evidence images for the game.

## Required Placeholder Images

Create the following placeholder images (recommended size: 512x512 pixels):

1. `placeholder_document.png` - Generic document/paper image
2. `placeholder_leak.png` - Water leak or damage image
3. `placeholder_jewelry.png` - Jewelry/bracelet image

## Evidence Images by Chapter

### Chapter 1
- exam_papers_c1 → placeholder_document.png
- water_source_c1 → placeholder_leak.png
- bracelet_c1 → placeholder_jewelry.png
- maintenance_log_c1 → placeholder_document.png
- witness_statement_c1 → placeholder_document.png

## Image Guidelines

- Use PNG format with transparency
- Recommended size: 512x512 pixels
- Keep file sizes reasonable (under 500KB each)
- Use clear, recognizable imagery
